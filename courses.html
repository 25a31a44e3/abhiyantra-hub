Module 2: NumPy Mastery

Array Programming

NumPy provides N-dimensional array objects and tools for working with these arrays:

  • ndarray: The core object (np.array([1,2,3]))
  • Broadcasting: Vectorized operations without loops
  • Universal Functions: Fast element-wise operations (np.sin, np.exp)
  • Indexing: Fancy indexing, boolean masks

Performance

NumPy arrays are faster than Python lists because:

  1. Contiguous memory allocation
  2. Vectorized operations in C
  3. Fixed-type elements

Machine Learning - Complete Theory & Practice

Module 1: Supervised Learning

Fundamentals

Machine learning algorithms that learn from labeled training data:

1. Linear Regression

Models the relationship between dependent (y) and independent variables (X):

y = β₀ + β₁X₁ + ... + βₙXₙ + ε

Cost Function: MSE = 1/n Σ(yᵢ - ŷᵢ)²

2. Logistic Regression

For binary classification (0/1):

σ(z) = 1 / (1 + e⁻ᶻ) where z = wᵀx

Cost Function: Cross-Entropy

3. Decision Trees

Recursive partitioning of feature space using:

  • Gini Impurity: 1 - Σ(pᵢ²)
  • Information Gain: H(parent) - Σ(child_weight * H(child))

Deep Learning - Complete Neural Networks Guide

Module 1: Neural Network Fundamentals

Perceptron

Basic building block with:

Output = f(w·x + b) where f is activation function

Backpropagation

Chain rule for gradient computation:

  1. Forward pass: Compute loss
  2. Backward pass: Compute gradients ∂L/∂w for all parameters
  3. Update weights: w = w - η(∂L/∂w)
